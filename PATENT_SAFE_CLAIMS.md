# Patent-Safe Claims & Technical Wording

This document provides precise terminology for use in patent filings, project forms, or formal demonstrations to avoid overclaiming while protecting the intellectual property of Collective MindGraph.

## 1. Safe Technical Claims
- "A local-first system configured to process technical Turkish audio data without external network reliance."
- "A dual-transcript processing architecture that concurrently maintains raw inference output and cleaned textual representations."
- "Heuristic-based extraction of organizational entities, including task identifiers and decision markers, from processed conversational data."
- "A traceable knowledge retrieval layer that maps extracted insights back to original temporal segments in a session."
- "Implementation of mandatory URL validation guards to enforce local-only execution perimeters."

## 2. Risky Claims to Avoid (The "Do Nots")
- **Do NOT claim**: "Fully autonomous meeting understanding."
- **Do NOT claim**: "100% accurate Turkish transcription."
- **Do NOT claim**: "Semantic reasoning across arbitrary graph edges."
- **Do NOT claim**: "Production-ready for high-noise environments."

## 3. Implemented vs. Planned Language
| Feature | Implemented Language | Planned Language |
| :--- | :--- | :--- |
| **Search** | "Keyword-based retrieval" | "Conceptual semantic retrieval" |
| **Memory** | "Basic graph-node persistence" | "Arbitrary relational graph edges" |
| **Hardware**| "Software MVP implementation" | "Dedicated hardware capture device" |
| **Logic** | "Deterministic heuristic matching" | "Multi-hop graph reasoning" |

## 4. Key Terminology for Privacy
Use these terms to describe the system's security model:
- **"Data Sovereignty"**: User data remains within the local infrastructure.
- **"Offline Perimeter"**: Strictly enforced boundary against internet egress.
- **"On-Premise Inference"**: Model execution occurs on local compute resources (CPU/GPU).

## 5. Hardware/Software Direction Wording
"The system architecture **can be configured to** integrate with specialized hardware components, including but not limited to far-field microphone arrays and dedicated local inference modules, for enhanced captured and processing."

## 7. Official Project Status
"The project is local MVP demo ready and product-integration ready for local-first Turkish transcription and keyword-based memory exploration. It is not yet a production-validated meeting intelligence platform."
