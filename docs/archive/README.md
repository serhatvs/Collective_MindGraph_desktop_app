# Documentation Archive

This directory contains historical material that is useful for provenance but is not authoritative for the current repository.

- `handovers/`: dated project snapshots, branch boundaries, and superseded work plans.
- `companion/`: retired companion-service documentation.
- `concept_modules/`: concept-stage module material.
- `old/`: other superseded documents retained for traceability.

Use `docs/README.md`, `docs/dev/ARCHITECTURE.md`, and `docs/dev/SETUP.md` for current guidance.
